# Guitar Synth MIDI Renderer

A Python project that generates guitar MIDI riffs and chord progressions, then renders them to WAV files using **FluidSynth** and custom SoundFonts. Perfect for music production, experimentation, or learning about MIDI synthesis.

---

## Features

- Generate **clean guitar riffs**, Grateful Dead-style riffs, and chord progressions in MIDI format.
- Render MIDI files to WAV with realistic guitar sounds using SoundFonts.
- Supports random tempos, scales, and note variations for creative output.
- Skips hidden macOS system files automatically.
- Provides detailed logging for errors during rendering.

---

## Requirements

1. **Python 3.10+**  
2. **FluidSynth**
   install for MAC using: brew install fluid-synth
3. Any free guitar soundfont
